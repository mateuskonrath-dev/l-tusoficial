## 🛠️ How to Use New Features

### Dark Mode
1. Click sun/moon icon in navbar
2. Preference saves automatically
3. System theme respected if not set

### PWA Installation
**Desktop:**
1. Click "Install" button in address bar
2. App opens in standalone window

**Mobile:**
1. Tap "Add to Home Screen" (iOS)
2. Tap "Install app" (Android)
3. App appears on home screen
